# AGENTS.md

## Project instructions

If you need to interact with Linear from this repo, read `skills/linear-mcporter/SKILL.md` first and follow it.

Use cases:
- listing or searching Brew issues
- creating or updating Linear issues
- adding status comments tied to code changes
- checking projects, documents, cycles, labels, or statuses

Prefer the repo-local MCPorter config at `config/mcporter.json` instead of relying on global config.
